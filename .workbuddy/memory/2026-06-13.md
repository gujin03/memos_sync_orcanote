## Memos 源码分析

- 研究了 [usememos/memos](https://github.com/usememos/memos) 项目的 API 实现
- 关键发现：ListMemos 支持 CEL 语法的 `filter` 参数，`created_ts`/`updated_ts` 是 Unix 秒级时间戳

## 服务端过滤优化

- **改动**：将 `fetchMemos()` 中的客户端日期过滤（全量拉取 → 逐条 filter）改为服务器端 CEL 过滤
- `startDate` → `created_ts >= <unix_seconds>`
- `lastSyncTime` → `updated_ts >= <unix_seconds>`
- **效果**：Memos 服务端直接返回过滤后的结果，减少网络传输和 API 请求次数
- 移除 18 行客户端过滤代码，Git 提交 `9ae2086`
